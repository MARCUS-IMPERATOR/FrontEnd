```
assets/

```

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MARCUS-IMPERATOR/FrontEnd.git
   cd FrontEnd
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run android
   ```
